功能：
======
文件：
-----
	文件的存储，新建，打开，保存，另存
画图:
-----
	可以绘制直线，曲线，矩形，圆形
文字：
------
	点击文字按钮输入要绘制的文字后在屏幕上拖拽生成
移动:
---------
	选中对象并进行移动
粗细：
------
	按<和>也就是,和.进行粗细的调整
大小：
-----
	按-(_)和=(+)调整直线的长度，图形和文字的大小
